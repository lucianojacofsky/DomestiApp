# DomestiApp

DomestiApp es una aplicación tipo marketplace de servicios domésticos (plomeros, electricistas, gasistas, etc.), inspirada en el modelo de Uber pero aplicada a profesionales del hogar.

## 🚀 Estado del proyecto
- Backend inicial con Node.js + Express
- Frontend web creado con React
- Configuración de `.gitignore` para excluir `node_modules`

## 📂 Estructura
-  /backend   → servidor Node.js 
-  /frontend  → aplicación web en React (y futura app móvil)

## 🔧 Tecnologías
- Node.js + Express
- MongoDB (próximamente)
- React (web)
- React Native (móvil, futuro)
- MercadoPago (pagos)

## 📌 Próximos pasos
- Definir modelos de datos (usuarios, trabajadores, servicios)
- Implementar autenticación y verificación con DNI
- Integrar pagos con MercadoPago
